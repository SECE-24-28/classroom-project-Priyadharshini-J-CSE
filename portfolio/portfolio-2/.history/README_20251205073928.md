# Priyadharshini J — Portfolio

Simple static portfolio built with plain HTML, CSS and JavaScript.

How to view
- Open `index.html` in your browser (double-click or use Live Server).

Files added
- `index.html` — main page
- `css/styles.css` — design and responsive rules
- `js/script.js` — small interactive behaviors (nav toggle, print)

Notes
- Use the "Print / Download Resume" button to create a PDF via the browser's print dialog.
- This site is intentionally dependency-free so it can be hosted anywhere (GitHub Pages, Netlify, or static web hosting).
